**II - How to use?**

Open Visual Studio Code from this directory:
```
booking_app_backend
├── controllers
├── data
├── node_modules
├── routes
├── .env
├── config
├── index
├── package.json
└── package-lock.json
```

Open new terminal, run: npm start

</br>

**III - Have error?**

</br>

Try this:
1. Delete node_modules
2. run: npm i
3. run: npm start
